This is an example project for the EthereumZuri.ch (2023) "Efficient cross-chain fuzzing" workshop.

## Setup

- Install Woke:
    - `pip3 install woke`
- Install node dependencies:
    - `npm ci`
- Generate pytypes:
    - `woke init pytypes`
